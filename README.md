# ESP32 Weather Station

Este repositório contém o projeto **ESP32 Weather Station**, que utiliza a placa ESP32 para construir uma estação meteorológica que coleta e monitora dados ambientais, como temperatura, umidade e pressão atmosférica. O sistema permite a captura dos dados através de sensores conectados ao ESP32 e seu envio para uma plataforma online para visualização em tempo real.

## Descrição do Projeto

O **ESP32 Weather Station** é projetado para capturar dados meteorológicos locais e transmiti-los via Wi-Fi, utilizando o ESP32. O projeto foca na simplicidade de construção e na eficiência para coletar dados ambientais e visualizar os resultados online.

## Funcionalidades

- Coleta de dados de temperatura, umidade e pressão atmosférica.
- Transmissão dos dados em tempo real via Wi-Fi.
- Integração com plataformas online para visualização remota dos dados.

## Tecnologias Utilizadas

- **ESP32**
- **Sensores de temperatura, umidade e pressão**
- **Arduino IDE**
- **Plataformas de visualização de dados (como ThingSpeak ou Blynk)**

## Estrutura do Repositório

- `src/`: Código principal para a coleta de dados e transmissão usando ESP32.
- `docs/`: Documentação detalhada sobre a configuração do hardware e software.
- `data/`: Exemplos de dados coletados.

## Como Executar

### Pré-requisitos

- Placa ESP32.
- Sensores de temperatura, umidade e pressão conectados ao ESP32.
- Arduino IDE configurado para ESP32.
- Conexão Wi-Fi para enviar os dados.

### Passos para execução

1. Clone o repositório:
   ```bash
   git clone https://github.com/Giovanacarmazio/Estacao-Met-ESP32.git
   ```

2. Carregue o código na placa ESP32 através do Arduino IDE.

3. Configure as credenciais de Wi-Fi e a plataforma de visualização de dados.

4. Monitore os dados em tempo real.
